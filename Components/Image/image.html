Image

Images are crucial for displaying visual information on an application. In order to only use the Image Component from Cascade UI in your project, copy and paste the below code in the head tag of your html document. 

                
    <link rel="stylesheet" href="https://cascadeUI.netlify.app/css/foundation.css" />
    <link rel="stylesheet" href="https://cascadeUI.netlify.app/Components/Image/image.css" />

Responsive Image

Responsive images can be used by applying the class name img-responsive. You can copy the html part from below code snippet. 

                
                    <img src="/assets/image.jpg" class="img-responsive">

Round Image

Round images can be used by applying the class name rounded-img. You can copy the html part from below code snippet.

                        
                            <div class="img-container">
                                <img src="/assets/image.jpg" class="rounded-img">
                            </div>
